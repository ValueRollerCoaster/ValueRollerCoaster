### HR's Role: Enabling Values to Be Lived

**HR's Strategic Mission**: Help coworkers **experience and live company values**, not just read them. Bridge the gap between abstract values and daily work reality.

**How HR Enables Values Living:**

1. **Make Values Tangible**
   - Help coworkers see how values apply to their work
   - Show different interpretations across departments
   - Make values actionable tools, not abstract concepts

2. **Facilitate Understanding**
   - Help employees understand company values through practical application
   - Show how values are interpreted differently
   - Build bridges between different perspectives

3. **Connect to Sales**
   - Help everyone understand how they contribute to sales
   - Show how values drive customer value
   - Demonstrate sales contribution through values

4. **Create Alignment**
   - Build shared values understanding
   - Improve cross-department communication
   - Strengthen organizational culture

## Troubleshooting

### Company Profile Issues

**Problem**: Changes not saving
- **Solution**: Ensure all required fields are filled
- **Solution**: Check for validation errors
- **Solution**: Refresh page and try again

**Problem**: AI not using updated profile
- **Solution**: Refresh AI Framework cache
- **Solution**: Regenerate value components
- **Solution**: Clear browser cache

### AI Framework Issues

**Problem**: Frameworks not generating
- **Solution**: Check company profile has industries configured
- **Solution**: Verify company profile is complete
- **Solution**: Refresh framework cache

**Problem**: Validation failing
- **Solution**: Check API keys are configured
- **Solution**: Verify company profile is complete
- **Solution**: Try refreshing framework first

### User Management Issues

**Problem**: Cannot delete user
- **Solution**: Ensure user is deactivated first
- **Solution**: Check for confirmation dialog
- **Solution**: Verify admin permissions

**Problem**: Data export failing
- **Solution**: Check database connection
- **Solution**: Verify user has data to export
- **Solution**: Try exporting individual user data

### General Issues

**Problem**: Admin tools not showing
- **Solution**: Verify you're logged in as `default_user`
- **Solution**: Check admin permissions
- **Solution**: Refresh page and login again

**Problem**: Changes not persisting
- **Solution**: Check for save confirmation messages
- **Solution**: Verify database connection
- **Solution**: Check browser console for errors

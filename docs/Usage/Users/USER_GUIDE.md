## Getting Started

### First Time Setup

1. **Company Setup** (if not done by admin)
   - Enter your company information
   - Configure business details
   - Set up your profile

2. **Define Your Values** (Critical Step)
   - Navigate to **Value Components** tab
   - Fill out value components across 4 categories
   - At least 90% completion required for persona generation

3. **Generate Your First Persona**
   - Go to **Persona Generator** tab
   - Enter a prospect's website URL
   - Click **Generate Buyer Persona**
   - Wait 5-8 minutes for analysis

## Value Components Tab

### Overview

This is where you **define your organization's values**‚Äîthe foundation for everything else. Values are organized into 4 main categories.

### What You Can Do

#### 1. View Summary
- **Sunburst Chart**: Visual overview of all your value components
- **Completion Status**: See how complete each category is
- **Distribution**: Understand value distribution across categories

#### 2. Edit Value Components

**Four Main Categories:**
- **Technical Value**: Quality, Performance, Innovation, Sustainability
- **Business Value**: Cost Savings, Revenue Growth, Efficiency Gains
- **Strategic Value**: Competitive Advantage, Risk Mitigation, Partnership Development
- **After Sales Value**: Customer Support, Maintenance, User Experience

**For Each Component:**
- **Enter Your Value**: Describe what you offer (be specific, not generic)
- **AI Generates Customer Benefit**: Automatically creates customer-focused benefit
- **Edit if Needed**: You can modify the AI-generated benefit
- **Save**: Changes are saved automatically

#### 3. Generate Missing Fields

**Button**: "‚ö° Generate Missing"
- **What it does**: Uses AI to fill in empty value component fields
- **When to use**: When you have some values but want to complete the rest
- **How it works**: AI analyzes your company profile and existing values to suggest new ones

#### 4. Regenerate All Components

**Button**: "üîÑ Regenerate All"
- **What it does**: Regenerates all value components from scratch using AI
- **When to use**: When you want to start fresh or get new AI suggestions
- **Warning**: This will replace existing values (you can edit them afterward)

### Tips for Value Components

- **Be Specific**: Instead of "We're innovative" ‚Üí "We deliver cutting-edge solutions that solve problems competitors can't"
- **Focus on Customer Value**: Describe what customers get, not just what you do
- **Use Real Examples**: Reference actual customer success stories
- **90% Completion Required**: You need at least 90% of components filled to generate personas

## Persona Generator Tab

### Overview

Generate comprehensive buyer personas by analyzing prospect websites. The AI analyzes the website and matches it with your value components.

### What You Can Do

#### 1. Generate a New Persona

**Step-by-Step:**
1. **Enter Website URL**: Paste the prospect's company website URL
2. **Select Industry** (optional): Choose from dropdown or leave blank
3. **Click "Generate Buyer Persona"**: Starts the 5-8 minute analysis process
4. **Watch Progress**: Progress is shown in the sidebar with current step

**What Happens:**
- Three AI models analyze the website (Gemini, ChatGPT, Sonar)
- Quality gates validate accuracy at 9 checkpoints
- Value alignment matches prospect needs with your values
- Comprehensive persona is generated

#### 2. View Persona Details

Once generated, you'll see **5 tabs** with detailed information:

**Tab 1: Company Overview**
- Company name, website, industry
- Products and services
- Business model
- Market position

**Tab 2: Profile**
- Pain points
- Goals and objectives
- Value drivers
- Strategic priorities

**Tab 3: Value Alignment** ‚≠ê **Most Important**
- **Alignment Matrix**: Shows how your values match their needs
- **Match Scores**: 0-100% for each alignment
- **Evidence**: Website quotes supporting each match
- **Conversation Starters**: Ready-to-use questions and talking points

**Tab 4: Industry Context**
- Market intelligence
- Industry trends
- Economic indicators
- Competitive landscape

**Tab 5: AI Insights**
- AI reasoning (chain of thought)
- Cross-model validation results
- Quality confidence scores
- Additional insights

#### 3. Save Persona

**Button**: "üíæ Save Persona"
- **What it does**: Saves the generated persona to the database
- **When to use**: After reviewing and confirming the persona is useful
- **Result**: Persona becomes searchable in Persona Search tab

#### 4. Clear Persona

**Button**: "üóëÔ∏è Clear Persona"
- **What it does**: Clears the current persona from view
- **When to use**: When you want to generate a new persona
- **Note**: This doesn't delete saved personas

### Tips for Persona Generation

- **Use Complete URLs**: Include `https://` or `http://`
- **Wait for Completion**: Don't close the browser during generation (5-8 minutes)
- **Review Alignment Matrix**: This is the most valuable output‚Äîshows how you fit their needs
- **Use Evidence**: Reference website quotes in your conversations
- **Check Confidence Scores**: Higher scores (8-10) mean more reliable results

## Persona Search Tab

### Overview

Search, view, and manage your saved buyer personas.

### What You Can Do

#### 1. Search Personas

**Search Bar**: "Search by company, website, or scan date"
- **Type to search**: Filters personas as you type
- **Searches**: Company name, website URL, scan date
- **Real-time**: Results update instantly

#### 2. Sort Personas

**Click column headers** to sort:
- **Company Name**: Alphabetical sorting
- **Website**: Alphabetical sorting
- **Scan Date**: Chronological sorting (newest/oldest first)

**Sort Direction**: Click again to reverse sort order

#### 3. View Persona Details

**Click "View Details"** button on any persona:
- Opens full persona view (same as in Persona Generator)
- All 5 tabs available (Company Overview, Profile, Value Alignment, etc.)
- Can navigate back to search

#### 4. Delete Personas

**Button**: "üóëÔ∏è Delete" on each persona
- **Confirmation**: Asks for confirmation before deleting
- **Permanent**: Deletion cannot be undone
- **Use carefully**: Only delete if you're sure

### Tips for Persona Search

- **Use Search**: Faster than scrolling through many personas
- **Sort by Date**: Find most recent personas quickly
- **Review Before Deleting**: Make sure you don't need the persona anymore

## Understanding the Output

### Alignment Matrix (Most Important)

**What it shows:**
- **Prospect Need**: What the prospect needs or values
- **Your Value Component**: Which of your values matches
- **Match Score**: 0-100% how well you match
- **Evidence**: Website quotes supporting the match
- **Conversation Starter**: Suggested question or talking point

**How to use:**
- **High scores (80-100%)**: Lead with these in conversations
- **Medium scores (50-79%)**: Use as supporting points
- **Low scores (0-49%)**: May indicate areas to develop or skip

### Confidence Scores

**What they mean:**
- **8-10**: High confidence, reliable results
- **5-7**: Medium confidence, use with validation
- **0-4**: Low confidence, do additional research

**Where to find:**
- In Persona Details ‚Üí AI Insights tab
- Shows overall confidence and per-step validation

## Troubleshooting

### Persona Generation Fails

**Check:**

**Try:**
- Use search function

## About the Technology

### Why Streamlit?

This app is built with **Streamlit**, a Python framework that makes it fast to build data applications. I chose it because:
- **Rapid Development**: Allows building functional apps quickly
- **Python-Based**: Easy to integrate with AI models and data processing
- **Good for Data Apps**: Perfect for displaying analysis results and interactive charts
- **Open Source**: Full control over the code and deployment

### Important Differences from Regular Web Apps

Streamlit works differently than traditional web applications. Here are some things to know:

#### Navigation

**Browser Back Button Doesn't Work**
- Unlike regular websites, the browser's back button won't navigate between pages
- **Use the sidebar buttons** instead to move between tabs
- The app uses internal navigation buttons, not browser URLs

**No Direct URL Access**
- You can't bookmark specific pages or share direct links to personas
- Everything is accessed through the app's navigation system
- Use the search function to find saved personas instead of URLs

#### User Management

**Custom User System**
- Streamlit doesn't have built-in user management
- I added a custom authentication system
- You need to log in to use the app
- Your data is tied to your user account

#### Session Management

**Custom Session Handling**
- Streamlit's default session management is basic
- I built custom session management to handle:
  - Progress tracking during persona generation
  - Saving your work across page refreshes
  - Managing multiple tasks

**Page Refreshes**
- If you refresh the page, you'll stay logged in
- Your current work is preserved in most cases
- However, if a persona generation is in progress, refreshing might reset the progress display (the task continues in the background)

#### Other Differences

**No Traditional Browser History**
- The browser history won't show individual pages you've visited
- Use the app's navigation buttons to move around
- The sidebar always shows your current location

**Interaction Model**
- Every button click or input change can trigger a page refresh
- This is normal Streamlit behavior‚Äîthe app reruns to update
- Don't worry if you see brief loading states‚Äîthis is expected

**Limited Browser Control**
- Some browser features (like right-click context menus) may work differently
- Copy/paste works normally
- Keyboard shortcuts may behave differently than on regular websites

### What This Means for You

**What to Do:**
- ‚úÖ Use sidebar buttons for navigation
- ‚úÖ Use the app's search function to find content
- ‚úÖ Wait for operations to complete before navigating away
- ‚úÖ Use the "Clear Tasks" button if you need to reset

**What Not to Do:**
- ‚ùå Don't use the browser back button (use sidebar instead)
- ‚ùå Don't try to bookmark specific pages
- ‚ùå Don't refresh during persona generation (task continues, but progress display may reset)
- ‚ùå Don't expect traditional browser history

These differences are by design‚ÄîStreamlit prioritizes rapid development and Python integration over traditional web app features. The app works well for its purpose, just with a slightly different interaction model.
